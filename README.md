# About Me:
With 12+ years of full-stack development experience, I've navigated the diverse landscapes of Fortune 500 giants and pioneering startups. My expertise spans the entire development lifecycle, with a focus on front-end excellence. I'm proficient in a diverse range of languages (JavaScript, Kotlin, Swift, etc.), frameworks (React, React Native, Node, Next), tools (Git, Docker, Xcode, Android Studio, etc.), and Cloud Services (AWS, GCP, etc.). My professional journey includes building systems from the ground up and leading successful development teams. I live by Steve Jobs' words, "Stay hungry, stay foolish," using them as guiding principles in my career. I continuously seek innovation while remaining open to change and improvement. This approach has driven success in production development and enhancing user engagement. Collaboration with exceptional individuals fuels my passion, and I find inspiration in inspiring others
